# テスト

アプリ本体は単一の `index.html`（ビルド無しのバニラJS）です。
ここには開発時の動作検証（ヘッドレスブラウザによるE2E）を置いています。
GitHub Pages で配信される本体には影響しません。

## 実行

```sh
npm i -D playwright
node tests/e2e.mjs
```

## 検証内容（`e2e.mjs`）

- 会計の記録（客数・合計・お釣り）
- 超過販売（仕入数より多く売れた場合に在庫がマイナスになり、レポートに警告として検出される）
- 会計済の部分修正（品目±が 売上・在庫・編集フラグ・操作ログ に反映される）
- 会計取消（客数・在庫・売上・取消件数 に反映される）
- データ整合性（品目合計＝取引合計）
- レポート（全7セクションの生成）
